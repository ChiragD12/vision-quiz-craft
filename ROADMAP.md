# Roadmap

## v2.0 (shipped)

- Deep-navy / gold design system.
- Revision Dashboard with streak ring (bronze/silver/gold/gem).
- Core subjects seeded (History, Polity, Geography, Economy, General Science,
  Art & Culture, Current Affairs) — hideable, undeletable.
- Custom subjects: create, rename, hide/unhide, delete.
- Subject → Topic → Knowledge drawer for management.
- Case-insensitive topic dedupe.
- Per-question quiz timer.
- Result screen with score, accuracy, questions, time, bookmarks, wrong.
- Trimmed Settings: key, import, export, erase, about.
- Full zero-regression parity with v1 features.

## v2.1 (next)

- Spaced-repetition surfacing on the dashboard ("5 due today").
- Explanation-only review mode (no scoring).
- Multi-topic quizzes (draw from N topics of one subject).
- Export a single quiz as PDF for offline printing.

## v2.2

- Rich note editor (markdown + inline images).
- On-device semantic search across all stored knowledge.
- Optional client-side encryption of the JSON export.

## Long term

- Native shells via Capacitor (iOS/Android) — same codebase.
- Optional peer sync (WebRTC, key-encrypted) — never a central server.
